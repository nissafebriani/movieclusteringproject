About the data:
The dataset consists of 291 data regarding the assessment of 5 film genres, namely horror, romcom, action, comedy, and fantasy. 


About the project:
This project focuses on clustering customers who give different ratings for each genre using k-mean clustering.

The language used in this project is R.

This project can be used as a reference. 


Library used:
tidyverse
cluster
factoextra
ggplot2
